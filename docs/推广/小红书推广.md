# 做了一个播客资料收集工具 🎧📚

汽车行业，想转 AI

小宇宙 1689h，啥播客都听

知行小酒馆、All-In、层次感……

每期少则几本，多则几十本书和文章

**问题是——听完就忘，想用的时候根本想不起来是哪期提到的**

---

**做了个工具 PodExtract**

🔗 粘贴播客链接
📋 自动提取：书籍、音乐、视频、人物、链接
💾 收藏到个人资料库

---

**用起来：**

📖 按播客回顾 → 这期提到哪些书？
🔍 按书籍检索 → 某本书在哪期听过？
👤 按人物查找 → 哪些期提到过这个人？

**听完不是终点，用起来才是**

---

技术：Next.js + Tailwind + 通义千问

个人项目，边用边改
