# PodExtract - 播客资料收集工具

## 这是什么项目

**PodExtract** 是一款播客资料收集工具。

粘贴播客链接，它能自动识别并整理出节目中提到的：书籍、音乐、视频、人物、相关链接。

然后——你可以**收藏保存**，变成你的个人播客资料库。

## 想解决什么问题

我自己是深度播客用户，每年在播客里听到的书估计有几百本。但问题在于：

- 听过就忘，根本不会去搜
- 想回顾一期节目提到的书，要重新翻 show notes
- 听了几十期后，根本不知道某本书是在哪期听到的

**PodExtract 想做的是：把播客里提到的知识，变成我可以检索、回顾、整理的个人资料库。**

## 效果示例

粘贴一期播客链接，例如「E229 盖伊·斯皮尔：一位价值投资者的谢幕」：

PodExtract 自动提取：

| 类型 | 结果 |
|------|------|
| 📚 书籍 | 《与巴菲特共进午餐时我顿悟的五个真理》《更富有、更睿智、更快乐》... |
| 👤 人物 | 盖伊·斯皮尔、巴菲特、芒格、李录、段永平... |
| 🔗 链接 | 相关播客节目、投资文章... |

你可以**一键收藏**，保存到自己的资料库。
