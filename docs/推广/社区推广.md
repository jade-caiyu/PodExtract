# PodExtract - 播客资料收集工具

## 这是什么

我叫 jade，汽车行业，想转 AI。

我这人啥播客都听，小宇宙 1689h，少说也得几百期。听完一期觉得"这本书不错"，转头就忘。等真的想用的时候，根本想不起来是哪期提到的。

知行小酒馆、All-In、层次感……每期节目里少则几本，多则几十本书和文章。以前都是听完算完，什么都没留下。

后来想，能不能把这些节目里提到的知识留下来，变成真的能用上的？

于是做了 PodExtract。

## 能干什么

粘贴播客链接，自动提取：书籍，音乐、视频、人物、相关链接。

提取完可以收藏，变成个人资料库。想用的时候：

- **按播客回顾**：这期节目提到哪些书？
- **按书籍检索**：某本书在哪期听过？
- **按人物查找**：这个人哪些期节目提到过？

**听完不是终点，用起来才是。**

## 技术细节

- 前端：Next.js 15 + Tailwind CSS v4
- 后端：Next.js API Routes
- LLM：阿里云通义千问
